
# Check Lotery

Uma aplicação que te ajuda a conferir resultados de loterias

## Stack utilizada

**Front-end:** React, Vite, React-Router

**Back-end:** Firebase


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/gabrielfpereira/loteria
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
